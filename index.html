HTML & CSS の開発環境を用意しよう！

自分のPCでWebページを開発する方法を学習しまよう

Githubでページを公開しよう！

Githubにログイン　⇒　右上の+から「New Repository」押下

Repository Name は基本的に　「アカウント名.github.io」

仮に「foo」と入力すればURLは「アカウント名.github.io/foo」となる

「Create Repository」押下　⇒　「Read Me」から詳細を入力　⇒　「Commit new file」

「Code」　⇒　「Add file」　⇒　どちらでもOK

「Settings」　⇒　「Pages」からサイトURLがわかる

URLに張り付ければページが表示される
